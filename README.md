# note Analytics Dashboard

noteクリエイター向けのアクセス解析BIツールです。CSVファイルをアップロードするだけで、フォロワー推移・ビュー・スキ・コメントなどのデータをグラフで可視化できます。

ログイン不要・バックエンド不要。HTMLファイル単体でブラウザ上で動作します。データは外部に送信されません。

---

## 機能

- **ダッシュボード** — フォロワー・総ビュー・総スキ・総コメントのKPIカード（前日比付き）、推移グラフ
- **記事パフォーマンス** — 記事別の累計・期間ビュー/スキ/コメント一覧、エンゲージメント率、複数記事の重ねグラフ表示
- **期間フィルタ** — 7日 / 30日 / 90日 / 全期間の切り替え
- **欠損日の自動補完** — 転記できなかった日はデータなしとして扱い、折れ線グラフは前後を繋いで表示
- **異常値の検出** — 累計値が減少している場合（転記ミスの可能性）に警告を表示
- **CSVテンプレートのダウンロード** — 正しいフォーマットのひな形を配布

## 使い方

### 1. ファイルをダウンロード

```
note_analytics.html
```

このファイル1つをダウンロードするだけで動作します。サーバーやインストールは不要です。

### 2. CSVを準備する

noteのダッシュボードを見ながら、以下の2種類のCSVファイルを手動で作成・更新します。

#### `summary.csv` — アカウント全体の日次データ

| カラム名 | 内容 |
|---|---|
| `date` | 記録日（例: `2026/04/15`） |
| `followers` | フォロワー累計数 |
| `total_views` | 全記事ビュー累計数 |
| `total_likes` | 全記事スキ累計数 |
| `total_comments` | 全記事コメント累計数 |

```csv
date,followers,total_views,total_likes,total_comments
2026/04/01,1000,5000,300,20
2026/04/02,1010,5120,312,21
```

#### `articles.csv` — 記事別の日次データ

| カラム名 | 内容 |
|---|---|
| `date` | 記録日（例: `2026/04/15`） |
| `article_title` | 記事タイトル |
| `views` | ビュー累計数 |
| `likes` | スキ累計数 |
| `comments` | コメント累計数 |

```csv
date,article_title,views,likes,comments
2026/04/01,記事タイトル,500,40,3
2026/04/01,記事タイトル2,200,10,1
2026/04/02,記事タイトル,520,43,3
2026/04/02,記事タイトル2,205,13,5
```

> **ポイント**
> - 日付は `YYYY/MM/DD` または `YYYY-MM-DD` のどちらでも受け付けます
> - 毎日1行ずつ末尾に追記していく運用です
> - 数値は**累計値**をそのまま記入してください。日次値はツール側で自動計算します
> - 2ファイルのどちらか一方だけでも動作します

### 3. ブラウザで開いてアップロード

1. `note_analytics.html` をブラウザで開く
2. 左メニューの「データ管理」からCSVをアップロード
3. ダッシュボード・記事パフォーマンスでデータを確認

## 動作環境

| 項目 | 内容 |
|---|---|
| 必要なもの | モダンブラウザのみ（Chrome / Firefox / Edge / Safari 最新版） |
| サーバー | 不要 |
| インターネット接続 | Chart.js・PapaParseをCDNから読み込むため初回起動時に必要 |
| データの保存先 | ブラウザのメモリのみ（外部送信なし・ページリロードでリセット） |

---

## 使用ライブラリ

- [Chart.js](https://www.chartjs.org/) v4.4.1 — グラフ描画
- [PapaParse](https://www.papaparse.com/) v5.4.1 — CSVパース
